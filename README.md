# DEV-CONNECT
DevConnect is a backend-driven social networking platform designed for developers to connect, share ideas, and interact with each other.

This project implements core social features such as user authentication, post creation, and commenting system. It uses JWT (JSON Web Token) for secure authentication and follows RESTful API architecture.

The backend is built using Node.js and Express, with MongoDB as the database. The project is structured in a clean and modular way with separate folders for routes, controllers, and configuration.

Features:
- User authentication (Signup/Login) using JWT
- Create, read, and manage posts
- Comment on posts
- Organized backend structure (MVC pattern)
- Secure API handling

Tech Stack:
- Node.js
- Express.js
- MongoDB
- JWT Authentication

This project is a great example of a beginner-to-intermediate level backend application and demonstrates real-world API development practices.
